Read Me

To run the Backend:
 
You will need .evn file which will need OPENAI_API_KEY

Step1:

   pip install requirements.txt

Step2:	

   python main.py


Endpoint Information:
	
	Currently I have 3 endpoints:

		-> /upload-commercial-invoice/
		-> /upload-fret/
		-> /upload-custom-declaration/
